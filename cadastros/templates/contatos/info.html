Sobre o site:

Este aplicativo foi desenvolvido com finalidade de estudo, para aprendizado e desenvolvimento de técnicas no Framework Django, SGBD PostgreSQL, HTML, CSS e Java Script.

O aplicativo consiste em um sitema de gerenciamento privativo de uma escola, onde são feitos os cadastros dos alunos, cursos, departamentos, grades, matrículas, etc. Há uma seção onde é feita consultas sobre os alunos e todos seus detalhes, qual curso está matriculado, disciplinas do curso e mais.
